Installation
-----------------------------------------------------------------

1] Copy all the installables i.e. SP-Android Folder on the "D:\" of your system

2] Copy the project in same SP-Android Folder
---------------------------------------------------------------

Open the project
----------------------------------------------------------------
1] Open Eclipse


2]click on File
	->Import
	->General
	->Existing Projects into workspace
	->Next
	->Select root directory and browse the project 
	  (D:\SP-Android\PersonalScheduling)
	->click Finish
	->project will be listed in Package Explorer of Eclipse

-----------------------------------------------------------------  
Execution of Project
-----------------------------------------------------------------
1] Open any of the java file available in
	->PersonalScheduling
	->src
	->impossible.is.nothing

Note:
	Make sure that you are not running the project when the xml file is opened.

2] Click on run icon or press (ctrl+F11)
	

3] When running project for the first time it willl ask for

	->Run As
	->select Android Application
	->open ConnectionDLL.cs
	->click OK
	->Let the project deploy
	->Unlock the lock
	->Execute the activities in project 
	
----------------------------------------------------------------

Note: 

1] The scheduling date should be in the format 24/3/2012

2] The scheduling time should be in the format 1:1 pm
